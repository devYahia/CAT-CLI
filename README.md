# CAT-CLI Solutions

This repository contains my solutions for the Backend Beginner Task.

## 1. To-Do List CLI (`todo.js`)
- Built using **Node.js**.
- **Features**: Add tasks, list tasks.
- **Data Persistence**: Uses `fs` module to save tasks into `tasks.json`.

## 2. Shortest String Challenge (`challenge-solution.js`)
- Solved using JavaScript's high-order function `reduce()` to efficiently find the minimum length string in an array.

## 3. SQL Challenge (`sql-challenge.sql`)
- SQL script to create a `students` table and retrieve records with grades between 90 and 100.
- **Tested with**: SQLite3.

## How to run:
- **To-Do App**: `node todo.js`
- **JS Challenge**: `node challenge-solution.js`
- **SQL Challenge**: `sqlite3 < sql-challenge.sql`
